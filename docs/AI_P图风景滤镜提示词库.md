# AI P图网站——风景滤镜提示词库

> 用途：适合自然风景、旅行照片、建筑、城市、海边、森林、雪景、夜景等。
>
> 建议统一追加：
>
> `保留原图地形、建筑、天空、道路、植物和主体构图，只调整色彩、天气氛围、光影、层次和整体画面风格。`

---

# 一、自然风景增强

## landscape_natural｜自然风景增强
```text
保留原始场景与构图，对整张风景照片进行自然增强，提升清晰度、通透感、色彩层次和光影细节，使天空、植物、建筑和地面更加自然耐看。
```

## landscape_clear｜清透风景
```text
将风景照片处理为清透明亮风格，提升空气感和远近层次，改善雾灰和暗沉，色彩干净自然，画面通透但不过度艳丽。
```

## landscape_high_dynamic｜高动态范围
```text
平衡天空高光与地面暗部，恢复云层、建筑、山体和植物细节，增强动态范围，保持真实自然，不要形成夸张 HDR 光晕。
```

## landscape_soft｜柔和自然风景
```text
降低风景照片中过硬的反差，使天空、地面和主体过渡更柔和，保留自然层次，画面舒服、安静、耐看。
```

## landscape_vivid_natural｜自然鲜活
```text
适度增强风景中的蓝天、绿植、建筑和环境色彩，使画面更鲜活有生命力，但保持色彩真实，不要过度饱和。
```

# 十一、统一负向提示词

```text
禁止改变地形和建筑结构，禁止新增无关物体，禁止删除主要景物，禁止天空与地面边缘错误，禁止建筑弯曲，禁止道路变形，禁止严重偏色，禁止高光过曝，禁止暗部死黑，禁止 HDR 光晕过重，禁止饱和度爆炸，禁止颗粒脏乱，禁止明显 AI 痕迹。
```
