# AI P图网站——电商滤镜提示词库

> 用途：适合商品主图、服装、美妆、珠宝、食品、数码、家居、家具、汽车用品等商业图片。
>
> 电商滤镜优先保证：商品外观、颜色、材质、结构、文字、Logo 和包装信息不被改变。
>
> 建议统一追加：
>
> `保持商品形状、结构、品牌、文字、Logo、包装、颜色和材质不变，只优化整体光影、背景氛围、色彩准确度和商业质感。`

---

# 一、通用商品图

## ecommerce_clean｜干净商品图
```text
将商品图片处理为干净清晰的电商风格，背景整洁，主体突出，光线均匀，颜色准确，材质和边缘清晰，适合商品详情页展示。
```

## ecommerce_bright｜明亮商品图
```text
提升商品图整体亮度和通透感，改善偏灰、偏暗和色偏，保留商品真实颜色和材质，画面明亮专业。
```

## ecommerce_main｜标准电商主图
```text
将图片处理为标准电商主图风格，主体居中突出，背景简洁，光线均匀，颜色准确，商品边缘、包装和文字清晰。
```

## ecommerce_high_end｜高级商业商品图
```text
将商品图处理为高级商业广告风格，光影精致，材质表现清晰，主体突出，色彩统一，整体具有高端品牌视觉质感。
```

## ecommerce_soft_light｜柔光商品图
```text
为商品加入柔和均匀的商业柔光，减少生硬阴影和高反光，保留商品结构和材质细节，整体自然高级。
```

## ecommerce_micro_contrast｜细节质感增强
```text
适度增强商品边缘、纹理、材质和包装细节，使画面更清晰立体，但不要过度锐化或改变原始材质。
```

# 十一、统一负向提示词

```text
禁止改变商品形状和结构，禁止改变品牌、Logo、文字和包装信息，禁止商品颜色失真，禁止新增配件，禁止删除商品部件，禁止材质错误，禁止高光过曝，禁止阴影死黑，禁止边缘模糊，禁止背景脏乱，禁止严重偏色，禁止文字乱码，禁止明显 AI 痕迹。
```
