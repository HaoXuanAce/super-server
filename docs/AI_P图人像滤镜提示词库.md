# AI P图网站——人像滤镜提示词库

> 用途：适合正式人像、写真、职业照、情侣照、婚纱照、街拍人像等。
>
> 前端只展示 `name`，后端根据 `id` 读取对应 `prompt` 进行拼接。
>
> 建议统一追加：
>
> `保持人物身份、五官、年龄、发型、服装、姿势和构图不变，只调整整体色彩、光影、氛围和质感。`

---

# 一、自然人像

## portrait_natural_clear｜自然清透
```text
将整张人像照片调整为自然清透风格，提升画面亮度、空气感和色彩通透度，肤色自然健康，阴影柔和，五官和皮肤细节真实，整体像高质量自然光人像摄影。
```

## portrait_clean_bright｜干净明亮
```text
将人像照片处理为干净明亮风格，提高整体明度和清爽感，改善暗沉和偏灰，保留自然肤色与面部层次，画面简洁、清晰、耐看。
```

## portrait_soft_light｜柔光人像
```text
为整张人像照片加入柔和自然的散射光效果，降低生硬阴影和高反差，保留面部立体感，画面温柔、细腻、自然，不要形成过度磨皮或明显光晕。
```

## portrait_high_end_natural｜高级自然感
```text
将人像处理为高级自然风格，色彩克制、光影均衡、层次丰富，肤色真实，质感细腻，像专业摄影师完成的商业人像后期。
```

## portrait_studio｜摄影棚人像
```text
将图片处理为专业摄影棚人像风格，主体轮廓清晰，光线柔和均匀，背景简洁，肤色自然，整体具有专业棚拍和商业精修质感。
```

## portrait_soft_contrast｜柔和低对比
```text
降低人像照片中过强的对比度，保留高光和暗部细节，让肤色、服装和背景过渡更加柔和，整体温润自然。
```

## portrait_micro_contrast｜微对比质感
```text
适度增强人像细节和局部明暗层次，使面部、头发和服装更加立体清晰，但避免过度锐化和皮肤粗糙。
```

# 十、统一负向提示词

```text
禁止改变人物身份，禁止改变五官和脸型，禁止改变年龄和性别，禁止新增无关人物，禁止服装和姿势变化，禁止严重偏色，禁止肤色异常，禁止过曝，禁止死黑，禁止过度磨皮，禁止蜡像感，禁止颗粒脏乱，禁止滤镜过重，禁止明显 AI 痕迹。
```
